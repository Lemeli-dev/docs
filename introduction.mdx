## What is Evidently?

Evidently helps evaluate, test, and monitor data and AI-powered systems.

* Evidently **open-source Python library** has over 25 million downloads. It provides 100+ evaluation metrics, a declarative API for conditional testing, and a lightweight visual interface to explore the results.

* **Evidently Cloud platform** offers a complete AI observability toolkit. It includes tracing, synthetic data generation, dataset management, eval orchestration, alerting and a no-code interface for domain experts to collaborate on AI quality.

Our goal: help teams build reliable AI systems and their performance throughout the lifecycle.
