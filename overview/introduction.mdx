If you are new to Evidently, this section gives an overview of the features, and answers popular questions.


Evidently helps evaluate, test, and monitor data and AI-powered systems. It supports:

* Generative systems: summarization, Q\&A, chatbots, RAGs, etc.

* Predictive ML models: classification, regression, ranking, recommendations.

* Data monitoring: data quality and data drift for text, tabular data, embeddings.

Evidently is available both as an open-source Python library and Evidently Cloud platform.
