You can use **Metrics** and Metric **Presets** to:

* Aggregate results of row-level text evaluations (see [descriptors](/docs/metrics/all_descriptors)).

* Assess data quality and data drift

* Evaluate quality on predictive ML tasks.

This is a reference page that shows available Metrics and Presets. Each Metric also includes a default visualization. For more details on implementations of the individual Metrics and their parameters, see linked pages.
