# Text Evals

Text Evals only apply to text columns. To compute a Descriptor for a single text column, use a `TextEvals` Preset. Read [docs](../tests-and-reports/text-descriptors.md).

You can also explicitly specify the Evidently Metric (e.g., `ColumnSummaryMetric`) to visualize the descriptor, or pick a [Test](all-tests.md) (e.g., `TestColumnValueMin`) to run validations.

## Descriptors: Text Patterns

Check for regular expression matches.

| Name                  | Description                                                                                                                                                                                                                                                                                    | Parameters                                                                                                                                                            |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **RegExp()**          | <ul><li>Checks if the column matches a specified regular expression.</li><li>Returns True/False for every input.</li><li>Example: `RegExp(reg_exp=r"^I")`</li></ul>                                                                                                                          | **Required:** <ul><li>`reg_exp`</li><li></ul>**Optional:** <ul><li> `display_name`</li></ul>                                                                                        |
| **BeginsWith()**      | <ul><li>Checks if the column begins with a specified combination.</li><li>Returns True/False for every input.</li><li>Example: `BeginsWith(prefix="How")`</li></ul>                                                                                                                            | <ul><li>**Required:** `prefix`</li><li>**Optional:** `display_name` </li><li> `case_sensitive = True` or `False`</li></ul>                                            |
| **EndsWith()**        | <ul><li>Checks if the column ends with a specified combination.</li><li>Returns True/False for every input.</li><li>Example: `EndsWith(suffix="Thank you.")`</li></ul>                                                                                                                         | <ul><li>**Required:** `suffix`</li><li>**Optional:** `display_name`</li><li>`case_sensitive = True` or `False`</li></ul>                                              |
| **Contains()**        | <ul><li>Checks if the column contains any or all specified items (e.g., competitor names).</li><li>Returns True/False for every input.</li><li>Example: `Contains(items=["medical leave"])`</li></ul>                                                                                          | <ul><li>**Required:** `items: List[str]`</li><li>**Optional:** `display_name` </li><li> `mode = any` or `all` </li><li> `case_sensitive = True` or `False`</li></ul>  |
| **DoesNotContain()**  | <ul><li>Checks if the column does not contain the specified items.</li><li>Returns True/False for every input.</li><li>Example: `DoesNotContain(items=["as a large language model"])`</li></ul>                                                                                                | <ul><li>**Required:** `items: List[str]`</li><li>**Optional:** `display_name` </li><li> `mode = all` </li><li> `case_sensitive = True` or `False`</li></ul>           |
| **IncludesWords()**   | <ul><li>Checks if the column includes **any** or **all** specified words. Considers only vocabulary words.</li><li>Returns True/False for every input.</li><li>Example: `IncludesWords(words_list=['booking', 'hotel', 'flight'])`</li></ul>                                                   | <ul><li>**Required:** `words_list: List[str]`</li><li>**Optional:** `display_name` </li><li> `mode = any` or `all` </li><li> `lemmatize = True` or `False`</li></ul>  |
| **ExcludesWords()**   | <ul><li>Checks if the column excludes all specified words. Considers only vocabulary words.</li><li>Returns True/False for every input.</li><li>Example: `ExcludesWords(words_list=['buy', 'sell', 'bet'])`</li></ul>                                                                          | <ul><li>**Required:** `words_list: List[str]`</li><li>**Optional:** `display_name` </li><li> `mode = all` </li><li> `lemmatize = True` or `False`</li></ul>           |
| **ItemMatch()**       | <ul><li>Checks if the column contains **any** or **all** specified items specific to each row, provided in a separate column.</li><li>Returns True/False for each row.</li><li>Example: `ItemMatch(with_column="expected")`</li></ul>                                                          | <ul><li>**Required:** `with_column: str`</li><li>**Optional:** `display_name` </li><li> `mode = all` or `any` </li><li> `case_sensitive = True` or `False`</li></ul>  |
| **ItemNoMatch()**     | <ul><li>Checks if the column excludes **all** specified items specific to each row, provided in a separate column.</li><li>Returns True/False for each row.</li><li>Example: `ItemNoMatch(with_column="forbidden")`</li></ul>                                                                  | <ul><li>**Required:** `with_column: str`</li><li>**Optional:** `display_name` </li><li> `mode = all`, `case_sensitive = True` or `False`</li></ul>                    |
| **WordMatch()**       | <ul><li>Checks if the column includes **any** or **all** specified words for each row, provided in a separate column. Considers only vocabulary words.</li><li>Returns True/False for every input.</li><li>Example: `WordMatch(with_column="expected")`</li></ul>                              | <ul><li>**Required:** `with_column: str`</li><li>**Optional:** `display_name` </li><li> `mode = any` or `all` </li><li> `lemmatize = True` or `False`</li></ul>       |
| **WordNoMatch()**     | <ul><li>Checks if the column excludes **all** specified words for each row, provided in a separate column. Considers only vocabulary words.</li><li>Returns True/False for every input.</li><li>Example: `WordNoMatch(with_column="forbidden")`</li></ul>                                      | <ul><li>**Required:** `with_column: str`</li><li>**Optional:** `display_name` </li><li> `mode = all` </li><li> `lemmatize = True` or `False`</li></ul>                |
| **ExactMatch()**      | <ul><li>Checks if the column contents matches between two provided columns.</li><li>Returns True/False for every input.</li><li>Example: `ExactMatch(with_column='reference')`</li></ul>                                                                                                       | <ul><li>**Required:** `with_column: str`</li><li>**Optional:** `display_name` </li></ul>                                                                              |
| **IsValidJSON()**     | <ul><li>Checks if the column contains valid JSON.</li><li>Returns True/False for every input.</li></ul>                                                                                                                                                                                        | <ul><li>**Optional:** `display_name`</li></ul>                                                                                                                        |
| **JSONSchemaMatch()** | <ul><li>Checks if the column contains a JSON object matching the expected schema. Supports exact or minimal matching.</li><li>Returns True/False for each row.</li><li>Example: `JSONSchemaMatch(expected_schema={"name": str, "age": int}, exact_match=False, validate_types=True)`</li></ul> | <ul><li>**Required:** `expected_schema: Dict[str, type]`</li><li>**Optional:** `exact_match = True` or `False` </li><li> `validate_types = True` or `False`</li></ul> |
| **JSONMatch()**       | <ul><li>Checks if the JSON contained in a column matches a JSON provided in a reference column. Compares key-value pairs irrespective of order. </li><li>Returns True/False for every input.</li><li>Example: `JSONMatch(with_column="column_2")`</li></ul>                                    | <ul><li>**Required:** `with_column: str`</li><li>**Optional:** `display_name`</li></ul>                                                                               |
| **ContainsLink()**    | <ul><li>Checks if the column contains at least one valid URL.</li><li>Returns True/False for each row.</li></ul>                                                                                                                                                                               | <ul><li>**Optional:** `display_name`</li></ul>                                                                                                                        |
| **IsValidPython()**   | <ul><li>Checks if the column contains valid Python code without syntax errors.</li><li>Returns True/False for every input.</li></ul>                                                                                                                                                           | <ul><li>**Optional:** `display_name`</li></ul>                                                                                                                        |
| **IsValidSQL()**      | <ul><li>Checks if the column contains a valid SQL query without executing the query.</li><li>Returns True/False for every input.</li></ul>                                                                                                                                                     | <ul><li>**Optional:** `display_name`</li></ul>                                                                                                                        |

## Descriptors: Text stats

Computes descriptive text statistics.

| Descriptor                                                                                                                                | Parameters                                               |
| ----------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------- |
| **TextLength()** - Measures the length of the text in symbols. Returns an absolute number.                                                | **Optional:** `display_name`                             |
| **OOV()** - Calculates the percentage of out-of-vocabulary words based on imported NLTK vocabulary. Returns a score on a scale: 0 to 100. | **Optional:** `display_name`, `ignore_words: Tuple = ()` |
| **NonLetterCharacterPercentage()** - Calculates the percentage of non-letter characters. Returns a score on a scale: 0 to 100.            | **Optional:** `display_name`                             |
| **SentenceCount()** - Counts the number of sentences in the text. Returns an absolute number.                                             | **Optional:** `display_name`                             |
| **WordCount()** - Counts the number of words in the text. Returns an absolute number.                                                     | **Optional:** `display_name`                             |

## Descriptors: LLM-based

Use external LLMs with an evaluation prompt to score text data.

| Descriptor                                                                                                                   | Parameters                                                                                                                                  |
| ---------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| **LLMEval()** - Scores the text using user-defined criteria.                                                                 | See [docs](../customization/llm_as_a_judge.md) for examples and parameters.                                                                 |
| **DeclineLLMEval()** - Detects texts containing a refusal or rejection. Returns a label (DECLINE or OK) or score.            | See [docs](../customization/llm_as_a_judge.md) for parameters.                                                                              |
| **PIILLMEval()** - Detects texts containing PII (Personally Identifiable Information). Returns a label (PII or OK) or score. | See [docs](../customization/llm_as_a_judge.md) for parameters.                                                                              |
| **NegativityLLMEval()** - Detects negative texts. Returns a label (NEGATIVE or POSITIVE) or score.                           | See [docs](../customization/llm_as_a_judge.md) for parameters.                                                                              |
| **BiasLLMEval()** - Detects biased texts. Returns a label (BIAS or OK) or score.                                             | See [docs](../customization/llm_as_a_judge.md) for parameters.                                                                              |
| **ToxicityLLMEval()** - Detects toxic texts. Returns a label (TOXICITY or OK) or score.                                      | See [docs](../customization/llm_as_a_judge.md) for parameters.                                                                              |
| **ContextQualityLLMEval()** - Evaluates if CONTEXT is VALID or INVALID. Returns a label (VALID or INVALID) or score.         | Run over the `context` column and pass the `question` column as a parameter. See [docs](../customization/llm_as_a_judge.md) for parameters. |

## Descriptors: Model-based

Use pre-trained machine learning models for evaluation.

| Descriptor                                                                                                                                                                                                                | Parameters                                                                         |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| **Semantic Similarity()** - Calculates pairwise semantic similarity between columns using a transformer model. Returns a score: 0 (different) to 1 (identical). Example use: `SemanticSimilarity(with_column="response")` | **Required:** `with_column`**Optional:** `display_name`                            |
| **Sentiment()** - Analyzes the sentiment of the text. Returns a score: -1 (negative) to 1 (positive).                                                                                                                     | **Optional:** `display_name`                                                       |
| **HuggingFaceModel()** - Scores the text using a user-selected HuggingFace model.                                                                                                                                         | See [docs](../customization/huggingface_descriptor.md) for examples.               |
| **HuggingFaceToxicityModel()** - Detects hate speech using a HuggingFace model. Returns predicted probability for the “hate” label. Scale: 0 to 1.                                                                        | **Optional:** `toxic_label`, `display_name`                                        |
| **BERTScore()** - Calculates similarity between two text columns based on token embeddings. Returns [BERTScore](https://arxiv.org/pdf/1904.09675) (F1 Score). Example use: `BERTScore(with_column="reference")`           | **Required:** `with_column`**Optional:** `model`, `tfidf_weighted`, `display_name` |
