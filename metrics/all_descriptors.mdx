# Text Evals

Text Evals only apply to text columns. To compute a Descriptor for a single text column, use a `TextEvals` Preset. Read [docs](../tests-and-reports/text-descriptors.md).

You can also explicitly specify the Evidently Metric (e.g., `ColumnSummaryMetric`) to visualize the descriptor, or pick a [Test](all-tests.md) (e.g., `TestColumnValueMin`) to run validations.

## Descriptors: Text Patterns

Check for regular expression matches.

| Name                | Description                                                                                                                                                     | Example                                                                                                                                                                                      | Parameters                                                                                                                              |
|---------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| **RegExp()**       | Matches text against any specified regular expression. Returns True/False for every input.                                                                       | `RegExp(reg_exp=r"^I")`                                                                                                                                                                      | <ul><li>**Required:** `reg_exp`</li><li>**Optional:** `display_name`</li></ul>                                                        |
| **BeginsWith()**   | Checks if the text begins with a specified combination. Returns True/False for every input.                                                                      | `BeginsWith(prefix="How")`                                                                                                                                                                   | <ul><li>**Required:** `prefix`</li><li>**Optional:** `display_name`, `case_sensitive`</li></ul>                                       |
| **EndsWith()**     | Checks if the text ends with a specified combination. Returns True/False for every input.                                                                        | `EndsWith(suffix="Thank you.")`                                                                                                                                                              | <ul><li>**Required:** `suffix`</li><li>**Optional:** `display_name`, `case_sensitive`</li></ul>                                       |
| **Contains()**     | Checks if the text contains any or all specified items (e.g., competitor names). Returns True/False for every input.                                             | `Contains(items=["medical leave"])`                                                                                                                                                          | <ul><li>**Required:** `items: List[str]`</li><li>**Optional:** `display_name`, `mode`, `case_sensitive`</li></ul>                     |
| **DoesNotContain()** | Checks if the text does not contain any or all specified items. Returns True/False for every input.                                                             | `DoesNotContain(items=["as a large language model"])`                                                                                                                                        | <ul><li>**Required:** `items: List[str]`</li><li>**Optional:** `display_name`, `mode`, `case_sensitive`</li></ul>                     |
| **IncludesWords()** | Checks if the text includes **any** or **all** specified words. Considers only vocabulary words. Returns True/False for every input.                              | `IncludesWords(words_list=['booking', 'hotel', 'flight'])`                                                                                                                                   | <ul><li>**Required:** `words_list: List[str]`</li><li>**Optional:** `display_name`, `mode`, `lemmatize`</li></ul>                     |
| **ExcludesWords()** | Checks if the text excludes all specified words. Considers only vocabulary words. Returns True/False for every input.                                            | `ExcludesWords(words_list=['buy', 'sell', 'bet'])`                                                                                                                                           | <ul><li>**Required:** `words_list: List[str]`</li><li>**Optional:** `display_name`, `mode`, `lemmatize`</li></ul>                     |
| **ItemMatch()**     | Checks whether the text contains **any** or **all** specified items specific to each row. Returns True/False for each row.                                       | `ItemMatch(with_column="expected")`                                                                                                                                                          | <ul><li>**Required:** `with_column: str`</li><li>**Optional:** `display_name`, `mode`, `case_sensitive`</li></ul>                     |
| **ItemNoMatch()**   | Checks whether the text excludes **any** or **all** specified items specific to each row. Returns True/False for each row.                                       | `ItemNoMatch(with_column="forbidden")`                                                                                                                                                       | <ul><li>**Required:** `with_column: str`</li><li>**Optional:** `display_name`, `mode`, `case_sensitive`</li></ul>                     |
| **WordMatch()**     | Checks whether the text includes **any** or **all** specified words for each row. Considers only vocabulary words. Returns True/False for every input.            | `WordMatch(with_column="expected")`                                                                                                                                                          | <ul><li>**Required:** `with_column: str`</li><li>**Optional:** `display_name`, `mode`, `lemmatize`</li></ul>                          |
| **WordNoMatch()**   | Checks whether the text excludes **any** or **all** specified words for each row. Considers only vocabulary words. Returns True/False for every input.            | `WordNoMatch(with_column="forbidden")`                                                                                                                                                       | <ul><li>**Required:** `with_column: str`</li><li>**Optional:** `display_name`, `mode`, `lemmatize`</li></ul>                          |
| **ExactMatch()**    | Checks if the text matches between two columns. Returns True/False for every input.                                                                             | `ExactMatch(with_column='reference')`                                                                                                                                                        | <ul><li>**Required:** `with_column: str`</li><li>**Optional:** `display_name`</li></ul>                                               |
| **IsValidJSON()**   | Checks if the text in a specified column is a valid JSON. Returns True/False for every input.                                                                    | N/A                                                                                                                                                                                          | <ul><li>**Optional:** `display_name`</li></ul>                                                                                        |
| **JSONSchemaMatch()** | Checks if the text contains a JSON object matching the expected schema. Supports exact or minimal matching. Returns True/False for each row.                     | `JSONSchemaMatch(expected_schema={"name": str, "age": int}, exact_match=False, validate_types=True)`                                                                                         | <ul><li>**Required:** `expected_schema: Dict[str, type]`</li><li>**Optional:** `exact_match`, `validate_types`</li></ul>             |
| **JSONMatch()**     | Compares two columns of a dataframe and checks whether the two objects in each row are matching JSONs. Returns True/False for every input.                        | `JSONMatch(with_column="column_2")`                                                                                                                                                          | <ul><li>**Required:** `with_column: str`</li><li>**Optional:** `display_name`</li></ul>                                               |
| **ContainsLink()**  | Checks if the text contains at least one valid URL. Returns True/False for each row.                                                                             | N/A                                                                                                                                                                                          | <ul><li>**Optional:** `display_name`</li></ul>                                                                                        |
| **IsValidPython()** | Checks if the text is valid Python code without syntax errors. Returns True/False for every input.                                                               | N/A                                                                                                                                                                                          | <ul><li>**Optional:** `display_name`</li></ul>                                                                                        |
| **IsValidSQL()**    | Checks if the text in a specified column is a valid SQL query without executing the query. Returns True/False for every input.                                    | N/A                                                                                                                                                                                          | <ul><li>**Optional:** `display_name`</li></ul>                                                                                        |


## Descriptors: Text stats

Computes descriptive text statistics.

| Descriptor                                                                                                                                | Parameters                                               |
| ----------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------- |
| **TextLength()** - Measures the length of the text in symbols. Returns an absolute number.                                                | **Optional:** `display_name`                             |
| **OOV()** - Calculates the percentage of out-of-vocabulary words based on imported NLTK vocabulary. Returns a score on a scale: 0 to 100. | **Optional:** `display_name`, `ignore_words: Tuple = ()` |
| **NonLetterCharacterPercentage()** - Calculates the percentage of non-letter characters. Returns a score on a scale: 0 to 100.            | **Optional:** `display_name`                             |
| **SentenceCount()** - Counts the number of sentences in the text. Returns an absolute number.                                             | **Optional:** `display_name`                             |
| **WordCount()** - Counts the number of words in the text. Returns an absolute number.                                                     | **Optional:** `display_name`                             |

## Descriptors: LLM-based

Use external LLMs with an evaluation prompt to score text data.

| Descriptor                                                                                                                   | Parameters                                                                                                                                  |
| ---------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| **LLMEval()** - Scores the text using user-defined criteria.                                                                 | See [docs](../customization/llm_as_a_judge.md) for examples and parameters.                                                                 |
| **DeclineLLMEval()** - Detects texts containing a refusal or rejection. Returns a label (DECLINE or OK) or score.            | See [docs](../customization/llm_as_a_judge.md) for parameters.                                                                              |
| **PIILLMEval()** - Detects texts containing PII (Personally Identifiable Information). Returns a label (PII or OK) or score. | See [docs](../customization/llm_as_a_judge.md) for parameters.                                                                              |
| **NegativityLLMEval()** - Detects negative texts. Returns a label (NEGATIVE or POSITIVE) or score.                           | See [docs](../customization/llm_as_a_judge.md) for parameters.                                                                              |
| **BiasLLMEval()** - Detects biased texts. Returns a label (BIAS or OK) or score.                                             | See [docs](../customization/llm_as_a_judge.md) for parameters.                                                                              |
| **ToxicityLLMEval()** - Detects toxic texts. Returns a label (TOXICITY or OK) or score.                                      | See [docs](../customization/llm_as_a_judge.md) for parameters.                                                                              |
| **ContextQualityLLMEval()** - Evaluates if CONTEXT is VALID or INVALID. Returns a label (VALID or INVALID) or score.         | Run over the `context` column and pass the `question` column as a parameter. See [docs](../customization/llm_as_a_judge.md) for parameters. |

## Descriptors: Model-based

Use pre-trained machine learning models for evaluation.

| Descriptor                                                                                                                                                                                                                | Parameters                                                                         |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| **Semantic Similarity()** - Calculates pairwise semantic similarity between columns using a transformer model. Returns a score: 0 (different) to 1 (identical). Example use: `SemanticSimilarity(with_column="response")` | **Required:** `with_column`**Optional:** `display_name`                            |
| **Sentiment()** - Analyzes the sentiment of the text. Returns a score: -1 (negative) to 1 (positive).                                                                                                                     | **Optional:** `display_name`                                                       |
| **HuggingFaceModel()** - Scores the text using a user-selected HuggingFace model.                                                                                                                                         | See [docs](../customization/huggingface_descriptor.md) for examples.               |
| **HuggingFaceToxicityModel()** - Detects hate speech using a HuggingFace model. Returns predicted probability for the “hate” label. Scale: 0 to 1.                                                                        | **Optional:** `toxic_label`, `display_name`                                        |
| **BERTScore()** - Calculates similarity between two text columns based on token embeddings. Returns [BERTScore](https://arxiv.org/pdf/1904.09675) (F1 Score). Example use: `BERTScore(with_column="reference")`           | **Required:** `with_column`**Optional:** `model`, `tfidf_weighted`, `display_name` |
